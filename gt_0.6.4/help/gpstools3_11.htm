Item Operation
Menu 　
Data 　
　 Time Span... 時間範囲指定ダイアログを表示し、時間範囲を設定します。
Satellites... 衛星リスト編集画面を表示し衛星リストを編集します。
Receivers... 受信機リスト編集画面を表示し受信機リストを編集します。
New Window 現在表示中のデータを再読込し、別ウインドウを開いて表示します。
Export Plot... 画像出力ダイアログを表示し、一連の画面を画像ファイルに保存します。
Map Area... 地図表示条件ダイアログを表示し地図表示条件を設定します。
Options... 表示オプションダイアログを表示し、表示オプションを変更します。
Plot 　
　 Satellite Ground Track 衛星直下点軌跡を地図上に表示します。
Receiver Position 受信機位置を地図上に表示します。
Coverage Area 衛星覆域を地図上に表示します。
Baseline Length 受信機間の基線長を地図上に表示します。
Voronoi Region Voronoi領域を地図上に表示します。
Receiver Skyplot 衛星方向軌跡をスカイプロット表示します。
Satellite Visibilities 衛星可視期間を表示します。
Elevation Angle by Sats 受信機毎衛星仰角を表示します。
Elevation Angle by Rcvs 衛星毎衛星仰角を表示します。
Satellite Eclipse Periods 衛星食期間を表示します。
Satellite-Sun Angle 衛星-太陽角をグラフ表示します。
DOP DOPをグラフ表示します。
Satellite 表示する衛星を選択します。
Receiver 表示する受信機を選択します。
(グラフ表示画面) グラフ下の時間軸近くでマウスの左ボタンをクリックすることによりカーソルが移動カーソルに変わり、左右にドラッグすることにより時間軸を移動することができます。
グラフ下の時間軸近くでマウスの右ボタンをクリックすることによりカーソルが拡大/縮小カーソル変わり、左右にドラッグすることにより時間軸を拡大/縮小することができます。
グラフ左の値軸近くでマウスの左ボタンをクリックすることによりカーソルが移動カーソルに変わり、上下にドラッグすることにより値軸を移動することができます。
グラフ左の値軸近くでマウスの右ボタンをクリックすることによりカーソルが拡大/縮小カーソル変わり、上下にドラッグすることにより値軸を拡大/縮小することができます。

Time Span Dialog

項目 操作
Start Time 開始日時を指定します。
End Time 終了日時を指定します。
Time Interval 時間間隔(秒)を指定します。
OK 設定に従い表示画面を更新します。
Cancel 設定変更を無効にして画面を閉じます。

Map Area Dialog

Options Dialog

Item Operation
Min Elevation Angle 最低仰角(度)を指定します。
Show Satellite Positoin in Map 衛星の軌跡だけでなく衛星位置も表示するか否か設定します。
・None : 衛星位置を表示しません。
・Start Position : 開始時刻における衛星位置を表示します。
・End Positin : 終了時刻における衛星位置を表示します。
Show Receiver Name 地図上に受信機名を表示するか否か設定します。
・OFF : 受信機名を表示しません。
・Code Name : 受信機IDを表示します。
・Full Name : 受信機名を表示します。
・Both Name : 受信機ID及び名前の両者を表示します。
Plot Line Style グラフ表示スタイルを選択します。
Plot Line Width / Marker Size グラフの線幅、マーカーサイズを指定します。
Receiver Position Color 地図上に表示する受信機位置の表示色を設定します。
Receiver Name Font 地図上に受信機名を表示する際に使用するフォントを指定します。
Input Navigation Message 入力する航法メッセージの種類を設定します。
Navigation Message Directory 入力する航法メッセージを格納したディレクトリパスを指定します。
OK 設定に従い表示画面を更新します。
Cancel 設定変更を無効にして画面を閉じます。
