GPS Precise Analysis Software GpsTools: Overview
Features | Functional Blocks | Processing Flow | References

Features

- Precise analysis with dual-frequency GPS/GNSS carrier-phase measurements
- Parameter estimation by Extended Kalman Filter/Smoother
- Estimation of receiver position, satellite orbit/clock, receiver clock, tropospheric parameters etc.
- Embedded precise measurement models and satellite motion models
- GUI tools to set parameter settings, execute analysis, plot results and for many utilities
- Common MATLAB libraries for user application programs
