Satellite Orbit Model Setting

Item Operation
Max Degree of Geopotential Set the maximum order/degree of the geopotential model.
Solar/Planetary Potentials Set the solar/planetary potential model.
- OFF : not include the other planetary potential
- Sun/Moon : include potential of the moon and the sun as point of mass
Solar Radiation Pressure Model Set the solar radiation pressure model.
- OFF : not include solar radiation pressure
- Simple : include simple (cannon ball) model
- ROCK4 : ROCK4/42
- GSPM.04 : GSPM.04b+estimation of scale, ybias parameters
- GSPM.04+A/C : GSPM.04b+estimation of scale, ybias, along-track and cross-track acceleration
- CODE(D/Y/B) : CODE RPR+estimation of D0, Y0 and B0
- CODE(D/Y/B/Z) : CODE RPR+estimation of D0, Y0, B0 and Z0
- CODE(D/Y/B/Z/X) : CODE RPR+estimation of D0, Y0, B0, Z0, X10 and X30
Eclipse Model Set the eclipse model.
- Cylindric : cylindric shadow of the earth
- Penumbra/Umbra : penumbra/umbra shadow of the earth and cylindric for the moon
Atmospheric Drag Model Set the atmospheric drag model.
- OFF : not include the atmospheric drag
Earth Tides Set the earth tides model.
- OFF : not include the earth tides
- Solid Earth Tide : include solid earth tide
- Solid/Ocean Tide : include solid earth tide, ocean tide and pole tide
Relativistic Effects Set the relativistic effect mode.
- OFF : not include relativistic effect
- IERS : include IERS Conventions 1996 corrections
Thrust Force Set the thruster force model.
- OFF : not include thruster force
Integration Step Set the step (sec) of numerical integration.
OK Make the changes effective and close the dialog.
Cancel Abandon all of the changes and close the dialog.
