Observation Data Editor Setting

Item Operation
L1 Frequency Set L1 carrier frequency (GHz). Usually set it 1.57542GHz.
L2 Frequency Set L2 carrier frequency (GHz). Usually set it 1.22760GHz.
Obs Time Tolerance Set observation time tolerance ttol (s). If ep-ttol/2 < rtime < ep+ttol/2, the observation data at the reception-time rtime are extracted as the epoch ep measurements.
Min Elevation Angle Set minimum elevation angle (deg).
Max Arc Gap Set maximum length of the arc gap (s). If there is longer outage than the gap in the observation data, the arc is separated at the outage.
Min Arc Length Set minimum length of an arc (s). If an arc is shorter than the length, it is excluded as invalid.
Min Arc Points Set minimum number of observation epochs in an arc. If an arc has less observation epochs than the number of points, it is excluded as invalid.
Noise of MW Code Set approximate standard deviation of MW (Melbourne Wubbena wide lane LC) noise (m) for cycle-slip detection.
Noise of Ion-Free Code Set approximate standard deviation of LC (ionospheric-free code LC) noise (m) for cycle-slip detection.
Slip Threshold of LG Set the threshold for cycle-slip detection by LG (geometry-free carrier-phase LC) jump (m). If a LG jump is over the threshold, the jump is recognized as a cycle-slip. If you set it 0, any cycle-slip is not detected by LG jump.
Slip Threshold of WL Set the threshold for cycle-slip detection by MW jump (cycle). If a WM jump is over the threshold, the jump is recognized as a cycle-slip. If you set it 0, any cycle-slip is not detected by MW jump.
Slip Threshold of MP1 Set the threshold for cycle-slip detection by MP1 LC jump (m). If a MP1 jump is over the threshold, the jump is recognized as a cycle-slip. If you set it 0, any cycle-slip is not detected by MP1 jump.
Slip Threshold of MP2 Set the threshold for cycle-slip detection by MP2 LC jump (m). If a MP2 jump is over the threshold, the jump is recognized as a cycle-slip. If you set it 0, any cycle-slip is not detected by MP2 jump.
Outlier Thres. of MW Set the condition for outlier rejection (sigma). If amplitude of MW LC exceed the standard deviation of MW LC in the arc multiplied by the threshold, the observation data is rejected as an outlier.
Ionos Smoothing Window
Moving Ave. Window 1 Set the number of observation data (points) to detect cycle-slip by MW WL, MP1, MP2 LC jump. The number is used as the window size of moving average/standard deviation before or after the detection point.
Moving Ave. Window 2 Set the number of observation data (points) to detect cycle-slip by MW WL, MP1, MP2 LC jump. The number is used as the window size of moving average/standard deviation before or after the detection point.
Slip Threshold Elevation Weighting Set the condition of the cycle-slip threshold weighting by elevation angle. If you set it on, the threshold is set to specified threshold / sin (elevation angle).
Repair Steered Clock-Jump Set whether the inconsistency between code and phase is repaired or not. The inconsistency is usually due to the receiver clock-jump derived from the intermittent clock steering in the receiver like Trimble, Ashtech etc. If setting it Adjust Phase, the phase observable are adjusted to the code. if setting it Adjust Code, the code observations are adjusted to the phase.
Repair Cycle-Slip Set the condition of cycle-slip fixing. If setting it ON, the cycle-slip is repaired and the arcs before and after the slip are connected to a continuous arc. If setting it OFF, these arcs are treated as separated arcs.
# of Points for Poly Fitting Set the number of observation data before or after the detected cycle-slip for polynomial fitting in order to repair cycle-slip.
Degree for Poly Fitting Set the order of polynomial for fitting to repair cycle-slip.
Separate Arc at Eclipse In/Out Set the condition of arc separation at the boundary of eclipse. If setting it on, carrier-phase ambiguity is initialized when the satellite put in or put out eclipse. If setting it off, the arc is treated as continuous one.Å@
