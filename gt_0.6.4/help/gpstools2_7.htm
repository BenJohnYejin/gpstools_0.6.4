Data Directories/Files Setting

Item Operation
Raw Observation Data Set the directory path of RINEX observation data.
Directory fill in the directory path or push [...] button and select a directory in the directory selection dialog.
Keywords in the directory path are replaced according to Keyword Replacement in Directory Path.
If the directory path is empty, Output Directory in Obs Editor/Parameter Estimator is used for the directory path. (This is the same for the following directory settings)
Navigation Message Set the directory path of RINEX navigation messages.
Satellite Ephemerides Set the directory path of satellite ephemerides, in case that the satellite orbits are fixed to predetermined values.
Satellite/Receiver Clocks Set the directory path of satellite and/or receiver clock data, in case that the satellite clocks and/or receiver clocks are fixed to predetermined values.
Receiver Positions Set the directory path of receiver (antenna) positions, in case that the receiver positions are fixed to predetermined values.
Earth Rotation Parameters Set the directory path of ERP (Earth Rotation Parameters), in case that ERP is fixed to predetermined value.
Tropos/Meteo Parameters Set the directory path of tropospheric or meteorological parameters data, in case that tropospheric or meteorological parameters are fixed to predetermined values.
Input Estimated Sat Params. Set the directory path of estimated satellite parameters, in case that satellite parameters are fixed to estimated values.
Input Estimated Rcv Params. Set the directory path of estimated receiver parameters, in case that receiver parameters are fixed to estimated values.
Satellite Antenna PCV Set the file path of satellite antenna PCV (phase center variation). Specify ANTEX format PCV file or satellite antenna PCV file (m-file) output by Plot Residuals/Statistics window.
Directly fill in the file path or push [...] button and select a file in the file selection dialog. By pushing [.] button, you can see the contents of the file by text editor.
If the file is empty, the default values are used for satellite PCV.
Receiver Antenna PCV Set the file path of receiver antenna PCV. Specify NGS (IGS_01.pcv) format or ANTEX format PCV file.
If the file is empty, receiver antenna PCV corrections is not applied.
Receiver Position Set the file path of receiver (antenna) position (m-file), in case that the receiver positions are specified as receiver position file.
Ocean Loading Parameter Set the file path of the ocean loading coefficients (BLQ format).
If the file path is empty, ocean loading correction is not applied.
Excluded Sats/Rcvs Set the file path of the excluded satellites and receivers.
If the file path is empty, any satellite or receiver are not excluded.
Phase Multipath Profile Set the file path of carrier-phase multipath profile. Specify the file output by Generate Multipath Profile.
If the file path is empty,carrier-phase multipath correction is not applied.
? Show information about Keyword replacement in directory of file paths
OK Enable changes and close dialog.
Cancel Abandon changes and close dialog.

Keyword Replacement in Directory or File Paths

Keyword Replaced string range
%P install directory path 　
%S station code (UPPER case) 　
%s station code (lower case) 　
%G station code (tail 4 chars) 　
%Y year (4 digits) 1970-9999
%y year (2 digits) 00-99
%m month 01-12
%d day 01-31
%n day of year 001-366
%W gpsweek no. 0000-9999
%D day of gpsweek 0,1,2,3,4,5,6
%M iers bulletin month 1,2,3,...
%O Output directory path
