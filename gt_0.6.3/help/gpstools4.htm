ファイル管理/操作ツール File Manager

項目 操作
\ ドライブ選択メニューを表示しドライブを選択します。
< 親ディレクトリに移動します。
> 子ディレクトリに移動します。
Directory ディレクトリ移動ダイアログを表示し指定されたディレクトリに移動します。
(Directory List) ディレクトリ一覧を表示します。
カレントディレクトリを移動するためにはディレクトリ一覧から移動したいディレクトリを選択してください。ディレクトリ一覧をダブルクリックすることにより指定ディレクトリの子ディレクトリに移動することができます。また「..」をダブルクリックすることにより親ディレクトリに移動することができます。
F フィルタダイアログを表示しフィルタを設定します。
Name, Ext, Size, Date/Time ファイル一覧を名前順、拡張子順、サイズ順、日時順に並べ替えます。ボタンを押す毎に正順、逆順を切り替えます。
(File List) ファイル一覧を表示します。
ファイル一覧に表示されたディレクトリ<directory>をダブルクリックすることによりディレクトリを移動することができます。ファイル一覧からファイルを選択しコマンドボタンを操作することによりファイルのコピー・移動・表示等の操作を行うことができます。
File 　
・New Directory... 新規ディレクトリ作成ダイアログを表示し作成したいディレクトリ名を入力することにより、カレントディレクトリに新規ディレクトリを作成することができます。
・Delete ファイル一覧からファイルまたはディレクトリを選択し、このボタンを押すことにより、指定ファイルまたはディレクトリを削除することができます。なおディレクトリを削除する場合、内容が空である必要があります。
・Uncompact ファイル一覧からファイルを選択し、このボタンを押すことにより圧縮ファイルを解凍・伸張することができます。対応している圧縮ファイルは以下の通りです。
・compress圧縮ファイル (*.Z)
・gzip圧縮ファイル (*.gz)
・Hatanaka-Compression RINEX観測データ (*.nnd)
・Compact ファイル一覧からファイルを選択し、このボタンを押すことによりファイルを圧縮することができます。
RINEX観測データはHatanaka-Compression RINEX形式に変換後、圧縮されます。圧縮後のファイル拡張子は.Zです。Uncompactメニューにより再度解凍・伸張することができます。
・Close ファイル管理・操作ツールを終了します。
Edit 　
・Copy... ファイル一覧からファイルを選択しメニューを実行することによりディレクトリ選択ダイアログが開きます。ディレクトリ選択ダイアログでコピー先ディレクトリを選択することにより、選択ファイルを指定ディレクトリにコピーすることができます。
・Move... ファイル一覧からファイルを選択しメニューを実行することによりディレクトリ選択ダイアログが開きます。ディレクトリ選択ダイアログで移動先ディレクトリを選択することにより、選択ファイルを指定ディレクトリに移動することができます。
・Rename... ファイル一覧からファイルを選択しメニューを実行することによりファイル名を変更することができます。
・Select All ファイル一覧を全選択します。
Filter ファイル一覧に表示するファイルの条件を指定します。空白の場合すべてのファイルが表示されます。
・Filter... フィルタダイアログを表示しフィルタを設定します。
・All (*.*) , ... 指定したパターンにフィルタを設定します。
View 　
・Update ファイル一覧を最新の内容に更新します。
・View... ファイル一覧からファイルを選択し、このボタンを押すことによりファイルの内容を表示することができます。対応しているファイル種別は以下の通りです。
・テキストファイル
・画像ファイル
・MATLAB matファイル (*.mat)
・HTMLファイル (*.html, *.htm)
テキストファイルの場合圧縮ファイルも直接内容を表示することができます。
Tools 　
・Download... データ/プロダクトダウンローダを起動します。
・Options オプションを設定します。

データ/プロダクトダウンローダ Data and Products Downloader

項目 操作
Data/Products (Source) ダウンロードしたいデータ、プロダクトのカテゴリ及び種別を指定します。
データ・プロダクト種別は同時に複数指定することができます。
Address ダウンロードするデータ、プロダクトのアドレスが表示されます。
アドレスのうちキーワードはダウンロードアドレス中のキーワード置換に従って置換されます。
Start Date/Time データ開始年月日時分を指定します。
End Date/Time データ終了年月日時分を指定します。
Sequence No データアドレスに連番が含まれている場合、連番の開始・終了番号を指定します。
Login User サーバのログインユーザ名を指定します。
anonymous ftpサーバの場合anonymousとして下さい。
Password サーバのログインパスワードを指定します。
anonymous ftpサーバの場合、自分のメールアドレスを指定して下さい。
Local Directory ダウンロードしたファイルを格納するディレクトリパスを指定します。
直接ディレクトリパスを入力するか「...」ボタンを押して表示されたディレクトリ選択ダイアログでディレクトリを選択してください。
ディレクトリパス中にキーワードが含まれている場合、ディレクトリパス中のキーワード置換に従い置換されます。指定ディレクトリが存在しない場合、自動的にディレクトリが生成されます。
Unzip/Uncompact Files ダウンロードするファイルが圧縮ファイルの場合、ダウンロード時に同時に解凍・伸張を行うかどうか選択します。
Skip Existing Files ダウンロードするファイルが既にダウンロード済の場合、そのダウンロードファイルをスキップするか否か選択します。
Abort on Download Error ダウンロードが失敗した場合に残りのファイルのダウンロードを中止するか否か選択します。
Stations... データアドレスに受信機名が含まれている場合に使用される受信機リストを表示します。
受信機リストを変更する場合は「Stations...」ボタンを押し受信機リスト編集画面で受信機リストを編集してください。
Files... ファイル管理/操作ツールを起動します。
Stations... 受信機リストを編集します。
Log... ダウンロードログを表示します。
Download/Abort 指定した条件でダウンロードを開始します。
ダウンロードを途中で中止する場合はAbortボタンを押して下さい。
Close データ・プロダクトダウンローダを終了します。

プロダクト生成 Generate Products

項目 操作
Start Date/Time 生成するプロダクトの開始日時を指定します。
End Date/Time 生成するプロダクトの終了日時を指定します。
Time Interval 生成するプロダクトの時間間隔 (秒) を指定します。
Processing Unit Time 入力するパラメータ推定値の解析単位時間 (hr)を指定します。
Product Type 生成するプロダクトの種別を指定します。
・Sat Ephemeris (SP3) : 衛星軌道
・Sat Clock (RINEX CLK) : 衛星時計
・Rcv Clock (RINEX CLK) : 受信機時計
・Rcv Position (SINEX) : 受信機位置
Product Source 入力するパラメータ推定値種別を指定します。
・Est(Smoothed) : Smoothed推定値
・Est(Forward) : Forward推定値
・Est(Backward) : Backward推定値
Generation Option プロダクト生成オプションを指定します。
・(空白) : オプション無し
・Include Velocity (SP3) : SP3出力で速度を含める。
Satellites/Receivers 生成するプロダクトの衛星/受信機リストを指定します。
Input Estimation Data Directory 入力するパラメータ推定値の格納されたディレクトリパスを指定します。
Output Products Directory 生成するプロダクトの出力先ディレクトリパスを指定します。
Generate 設定された条件でプロダクトを生成します。
Close 画面を閉じます。

項目 操作
Start Date/Time 生成する可降水量の開始日時を指定します。
End Date/Time 生成する可降水量の終了日時を指定します。
Time Interval 生成する可降水量の時間間隔 (秒) を指定します。

Output Products Directory 生成する可降水量の出力先ディレクトリパスを指定します。
Generate 設定された条件で可降水量を生成します。

海洋荷重潮汐係数生成 Generate Ocean Loading Parameters

項目 操作
Receivers 海洋荷重潮汐パラメータを生成する受信機リストを指定します。
「...」ボタンを押して表示される受信機リスト編集画面で受信機を指定して下さい。
Receiver Position 受信機位置データの種別を指定します。
Day of Position 受信機位置の日付を指定します。
Processing Unit Time 受信機位置の解析単位時間(hr)を指定します。
GOTIC2 Directory 海洋潮汐予測プログラムGOTIC2のインストールディレクトリを指定します。
gotic2.exeが格納されたディレクトリを指定して下さい。
Receiver Position Directory 受信機位置データの格納されたディレクトリを指定します。
Output Ocean Loading Parameter File 出力する海洋荷重パラメータファイルのファイルパスを指定します。
Generate 設定された条件で海洋荷重潮汐パラメータを生成します。
Close 海洋荷重潮汐パラメータ生成画面を閉じます。

マルチパスプロファイル推定 Generate Multipath Profile

項目 操作
Start Date/Time 入力する残差データの開始日時を指定します。
End Date/Time 入力する残差データの終了日時を指定します。
Time Interval 入力する残差データのの時間間隔 (秒) を指定します。
Processing Unit Time 入力する残差データの解析単位時間 (hr)を指定します。
Estimation Direction 入力する残差データの推定方向(Forward/Backward)を指定します。
Stations マルチパスプロファイルを生成する受信機リストを指定します。
Estimation Data Directory 入力する残差データが格納されたディレクトリパスを指定します。
Navigation Messages Directory 航法メッセージファイルが格納されたディレクトリパスを指定します。
Output Multipath Profile Directory 生成するマルチパスプロファイルの出力先ディレクトリパスを指定します。
Generate 設定された条件でマルチパスプロファイルを生成します。
Close 画面を閉じます。
