推定/固定パラメータ設定 Estimated or Fixed Parameter Settings

項目 操作
Satellite 衛星を指定して下さい。
ALLを指定した場合個々に設定した衛星以外のすべての衛星に適用されます。
Orbit 衛星軌道を推定するか固定するか指定します。
・(空白) : 放送暦の値に固定します。
・<estimated> : 衛星軌道を推定します。
・IGS Final : IGS最終暦の値に固定します。
・IGS Rapid : IGS速報暦の値に固定します。
・IGS URapid : IGS超速報暦の値に固定します。
・IGS URapid(Pred) : IGS超速報暦(予報値)の値に固定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothed推定値に固定します。
・ephs : 精密暦補間で生成した補間後精密暦の値に固定します。
・CODE, EMR, ESA, GFZ, JPL, MIT : それぞれ指定したIGS AC解の値に固定します。
・CODE Rapid : CODE Rapid暦の値に固定します。
Clock 衛星時計を推定するか固定するか指定します。
・(空白) : 放送暦の値に固定します。
・<estimated> : 衛星軌道を推定します。
・IGS Final : IGS最終暦の値に固定します。
・ IGS Rapid : IGS速報暦の値に固定します。
・ IGS URapid : IGS超速報暦の値に固定します。
・ IGS URapid(Pred) : IGS超速報暦(予報値)の値に固定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothed推定値に固定します。
・CODE, EMR, ESA, GFZ, JPL, MIT : それぞれ指定したIGS AC解の値に固定します。
・IGS/CODE : IGS Final時計をCODE時計で補間した推定値に固定します。
・CODE Rapid : CODE Rapid時計推定値に固定します。
・IGR/CODE Rapid : IGS Rapid時計をCODE Rapid時計で補間した推定値に固定します。
SRP Prams 太陽輻射圧パラメータを固定するか指定します。
・(空白) : デフォルト値または指定値に固定します。
・<estimated> : 太陽輻射圧パラメータを推定します。
なお衛星軌道を推定しない場合本設定は無視されます。
Receiver 受信機を指定して下さい。
ALLを指定した場合個々に設定した受信機以外のすべての受信機に適用されます。
Position 受信機位置を推定するか固定するか指定します。
・(空白) : 単独測位解に固定します。
・<estimated> : 受信機位置を推定します。
・IGS Final : IGS Final解位置に固定します。
・ITRF2000 : ITRF2000座標値に固定します。
・ITRF97 : ITRF97座標値に固定します。
・GSIPOS : GSI推定値(F1/F2)に固定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothedd推定値に固定します。
・IGS00 : IGS00座標値に固定します。
・IGb00 : IGb00座標値に固定します。
・Parameter File : 受信機位置パラメータファイルで指定した値に固定します。使用する受信機位置パラメータファイルはデータディレクトリ/ファイル設定画面で設定してください。
Clock 観測局時計を推定するか固定するか指定します。
・(空白) : コードによる単独測位で推定した値に固定します。
・<estimated> : 観測局時計を推定します。
・IGS Final : IGS最終暦の値に固定します。
・ＩGS Rapid : IGS速報暦の値に固定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothed推定値に固定します。
・CODE, EMR, ESA, GFZ, JPL, MIT : それぞれ指定したIGS AC解の値に固定します。
・IGS/CODE : IGS Final時計をCODE時計で補間した推定値に固定します。
・CODE Rapid : CODE Rapid時計推定値に固定します。
・IGR/CODE Rapid : IGS Rapid時計をCODE Rapid時計で補間した推定値に固定します。
Tropos. ZTD/Grad 対流圏遅延(天頂全遅延)を推定するか固定するか指定します。
・(空白) : アプリオリ対流圏遅延モデルによる推定値に固定します。
・<estimated> : 対流圏遅延量を推定します。推定・観測モデル設定画面で対流圏遅延勾配モデルが設定されている場合天頂全遅延及び対流圏遅延勾配を推定します。
・IGS ZPD : IGS Final 天頂全遅延(daily解)解に固定します。
・IGS ZPD Monthly : IGS Final 天頂全遅延(monthly解)に固定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothed推定値に固定します。
・JMA MSM Online : 気象庁数値予報モデル(MSM)から推定した天頂全遅延推定値に固定します。
Phase Bias 搬送波位相バイアスを推定するか固定するか指定します。
・(空白) : 搬送波位相バイアスを推定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothed推定値に固定します。
ERP 地球回転パラメータを推定するか固定するか指定します。
・(空白) : 値を0に固定します。
・<estimated> : 地球回転パラメータを推定します。
・IGS Final : IGS最終暦の地球回転パラメータの値に固定します。
・IGS Rapid : IGS速報暦の地球回転パラメータの値に固定します。
・IERS BulB : IERS Bulletin Bの値に固定します。
・IERS C04 : IERS C04の値に固定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothed推定値に固定します。
・CODE, EMR, ESA, GFZ, JPL, MIT : それぞれ指定したIGS AC解の値に固定します。
Geocent 地球中心オフセットを推定するか固定するか指定します。
・(空白) : 値を0に固定します。
・<estimated> : 地球中心オフセットを推定します。
Initial ERP 地球回転パラメータを推定する場合に地球回転パラメータ初期値を指定します。
・(空白) : 値を0に固定します。
・<estimated> : 地球回転パラメータを推定します。
・IGS Final : IGS最終暦の地球回転パラメータの値に固定します。
・IGS Rapid : IGS速報暦の地球回転パラメータの値に固定します。
・IERS BulB : IERS Bulletin Bの値に固定します。
・IERS C04 : IERS C04の値に固定します。
・Est(Forward) : Forward推定値に固定します。
・Est(Backward) : Backward推定値に固定します。
・Est(Smoothed) : Smoothed推定値に固定します。
・CODE, EMR, ESA, GFZ, JPL, MIT : それぞれ指定したIGS AC解の値に固定します。
Input Unit Time 推定値を入力する場合の推定時の推定単位時間 (hr) を指定します。
Day of Position 受信機位置を入力する場合の位置座標の日付を指定します。
空白の場合推定開始日の位置座標が使用されます。
Interp Clock 時計を入力する場合に入力時計の補間を行うかどうか指定します。
ONの場合入力時計は推定時間間隔に線形補間されます。
OK 設定変更を有効にして画面を閉じます。
Cancel 設定変更を無効にして画面を閉じます。
