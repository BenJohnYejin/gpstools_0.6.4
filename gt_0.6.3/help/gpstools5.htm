出力ファイル Output Files

ファイル名 形式 内容
obsc_{rcv}_yyyymmddhh.mat mat-flle 編集済観測データ
eph{fb}_{sat}_yyyymmddhh.mat mat-flle 衛星軌道推定結果
clk{fb}_{sat}_yyyymmddhh.mat mat-flle 衛星時計推定結果
pos{fb}_{rcv}_yyyymmddhh.mat mat-flle 受信機位置推定結果
clk{fb}_{rcv}_yyyymmddhh.mat mat-flle 受信機時計推定結果
zpd{fb}_{rcv}_yyyymmddhh.mat mat-flle 対流圏パラメータ推定結果
bcp{fb}_{rcv}_yyyymmddhh.mat mat-flle 搬送波位相バイアス推定結果
res{fb}_{rcv}_yyyymmddhh.mat mat-flle 残差・統計情報
{rcv} : receiver name, {sat} : satellite name (GPS01=GPS PRN1,GPS02=GPS PRN2,...),
{fb}: estimation direction (f=forward, b=backward),
yyyy : year (4digit), mm : month, dd : day *, hh : hour **

* 2日以上にまたがったデータの場合先頭データの受信日付によりファイル名がつけられる。
** 解析単位時間(Processing/Estimation Unit Time) (hr)の整数倍の値がつけられる。
例えば解析時間範囲が2004/12/3 4:00-9:30、解析単位時間3 hr、時間間隔1分の場合、*2004120303.matに4:00-5:59のデータ、*2004120306.matに6:00-8:59のデータ、*2004120309.matに9:00-9:30のデータが格納される。

編集済観測データファイル形式 File Format of Clean Observation Data

変数名

内容

備考

epoch

start epoch [year,month,day,hour,min,sec]

 　

time

time vector relative to epoch (sec)

 　

sats

satellite list (cell array)

 　

rcv

receiver name

 　

data

observation data
data(n,1): time(n) L1 carrier phase (cycle)
data(n,2): time(n) L2 carrier phase (cycle)
data(n,3): time(n) L1 pseudorange (m)
data(n,4): time(n) L2 pseudorange (m)
data(n,5): time(n) LC smoothed code (m)
data(n,6): time(n) L1 smoothed code (m)
data(n,7): time(n) L2 smoothed code (m)

 　

index

observation data index [s,0,f;...]
(s=satellite index,f=1:arc-start,2:arc-end)

 　

rstat

receiver states
rstat(n,1): time relative to epoch (sec)
rstat(n,2:7):receiver pos/vel (m,m/sec)
rstat(n,8): receiver clock bias (m)

 　

azel

satellite azimath/elevation angle (rad)
azel(n,1): satellite azimath angle (rad)
azel(n,2): satellite elevation angle (rad)

 　

slip

cycle-slip positions [t,s,f,...]
(t=time, s=satellite index, f=1:mw, 2:gf, 3:if, 4:td)

 　

rpos

receiver approx. position (m)(ecef)[x;y;z]

 　

adel

receiver antenna delta(up/east/north) (m)

 　

atype

receiver antenna type

 　

rtype

receiver model type
