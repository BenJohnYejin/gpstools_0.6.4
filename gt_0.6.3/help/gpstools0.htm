システム要件 System Requirements

項目 仕様
オペレーティングシステム Windows 2000、Windows XP
ソフトウェア環境 MATLAB 6.5.1 or 7.0.1以上
支援ソフトウェア 　
・データ圧縮・解凍 gzip
・圧縮RINEX生成・伸張 crx2rnx, rnx2crx
・海洋潮汐予測プログラム GOTIC2 (海洋荷重係数生成)
メモリ容量 512MByte以上

観測データ編集 Observation Data Editor

項目 仕様
入力観測データ RINEX 2.10 (高レートRINEX含む)
擬似距離コード P1+P2 またはC1+P2+P1-C1バイアス補正
P1-C1バイアス IGS解析センタ標準パラメータ
サイクルスリップ検出 幾何学フリー線形結合変化率 (電離層遅延変化率)
Melbourne-Wubbena Wide-Lane変化率 : Moving Average、Moving Standard Deviation
電離層フリー搬送波位相-擬似距離線形結合変化率
MP1, MP2 線形結合変化率
サイクルスリップ補修 Melbourne-Wubbenaアーク平均差によるスリップ量推定
幾何学フリーn次多項式FITによるスリップ量推定
アウトライア検出・除去 Melbourne-Wubbena 標準偏差による。
クロックジャンプ クロックジャンプ検出、補修
コードスムージング 搬送波位相観測データによるL1, L2コードスムージング
アーク分割 アーク分割及び指定サンプリング時間に従ったデータ抽出
編集済観測データ出力 MATLABバイナリ形式

パラメータ推定 Parameter Estimator

項目 仕様
推定方式 拡張カルマンフィルタ/スムーザ (EKF/S)
推定パス 1パス、2パス、3パス
推定可能パラメータ 受信機位置座標 (地球固定座標系)
対流圏天頂全遅延 (ZTD) 及び水平勾配 (North/East/2次項成分)
衛星・受信機時計誤差 (時計基準局相対時計バイアス・ドリフト)
衛星軌道 (衛星重心位置・速度)
衛星太陽輻射圧 (SRP) モデルパラメータ
地球回転パラメータ (ERP : Xp, Yp, UT1-UTC)
地球中心オフセット (Geocenter Offset : dx, dy, dz, Scale)
観測モデル 　
・基本観測量 電離層フリー搬送波位相
・差分観測量 ゼロ差/PPP、二重差
・測距モデル 地球重力を考慮した衛星運動二次近似及びLight Time Iteration
・衛星時計バイアス 推定または入力
・受信機時計バイアス 放送暦及び擬似距離観測値による単独測位で一次推定
・衛星アンテナオフセット Block II/IIA : X=0.279, Y=0, Z=1.023 m, Block IIR : X=0, Y=0, Z=0m (衛星固定座標系)
衛星アンテナ位相中心モデル入力可能 (ANTEX)
・受信アンテナオフセット 受信機アンテナ位相中心モデル (IGS_01.PCVまたはANTEX)
・対流圏遅延 A prioriモデル : Saastamoinen
マッピング関数 : COSZ, NMF (Niell Mapping Function)、またはGMF (Global Mapping Function)
全遅延モデル :
Mdry(EL)*Tropdry+Mwet(EL)(1+(Gnorth*cosAZ+Geast*sinAZ)/tanEL)*(ZTD-Tropdry)
(Mdry,Mwet : マッピング関数dry,wet項, Tropdry: A prioriモデルdry項、Gnorth,Geast : 水平勾配 North,East成分, AZ,EL: 方位仰角)、2次勾配項推定可能。
・電離層遅延　 一次項 : 電離層フリー線形結合で除去
高次項 : なし
・観測点位置変位
　(Site-Displacement)		 固体地球潮汐 :
IERS Conventions 1996 Ch.7
Step1 : in-phase degree2 and 3, out-of-phase, latitude dependence
Step2 : contribution from the diurnal band : K1 only
海洋荷重 :
IERS Conventions 1996 Ch.7
11分潮モデル (M2,S2,N2,K2,K1,O1,P1,Q1,Mf,Mm,Ssa)
モデル係数は任意の海洋潮汐モデル (BLQ形式)を入力可能
極運動潮汐 : IERS Conventions 1996 Ch.7
永年変位項 : 除去可能
大気荷重 : なし
・相対論効果 衛星時計及びBending Effect
・Phase-Windup効果 Wuモデル
・マルチパス 搬送波位相マルチパスプロファイルによる補正
Sidereal Filterによる補正
・観測雑音 天頂観測雑音 : ユーザ定義
仰角依存性 : sqrt(1+f(1/sin(EL)^2-1))
・搬送波位相バイアス 一次推定 : アーク毎に電離層フリー擬似距離により一次推定
推定モデル : アーク毎に電離層フリー搬送波位相バイアス残差推定 (Float解)
バイアス整数化 : なし
衛星運動モデル 　
・地球重力モデル JGM-3、最大12 次
・他天体重力　 月及び太陽 (質点)
・天体暦 JPL DE/LE405
・食モデル 円筒モデルまたは半影・本影モデル (地球及び月)
・太陽輻射圧 (SRP) モデル 単純モデル (Cannonballモデル)+Scaleパラメータ
ROCK4/42 (T20,T30)+Scale,Ybiasパラメータ
GSPM.04bモデル+Scale,Ybias,Along-Track,Cross-Track加速度パラメータ
CODE RPRモデル (D0,Y0,B0,Z0,X10,X30パラメータ)
・地球潮汐効果 地球固体潮汐 :
IERS Conventions 1996 Ch.6, frequency independent項2,3次、frequency dependent項K1 only
海洋潮汐 : IERS Conventions 1996 Ch.6 主要8分潮 2,3次 (CSR2.0)
極運動潮汐 : IERS Conventions 1996 Ch.6 ΔC21,ΔS21
・相対論効果 IERS Conventions 1996 Ch.11
・地球アルベド なし
・大気抵抗 なし
・任意加速度ΔV パラメータ入力可能
・数値積分 固定ステップ4次Runge-Kuttaまたは可変ステップ可変次数Adams-Bashforth-Moulton
状態遷移モデル 　
・衛星・受信機時計 White-Noiseまたは1次Gauss-Marcov過程
・対流圏遅延パラメータ Random-Walk過程または1次Gauss-Marcov過程
・地球回転パラメータ Random-Walk過程
・地球中心オフセット Random-Walk過程
・観測局位置 固定またはRandom-Walk過程(Static), またはWhite-Noise過程 (Kinematic)
・搬送波位相バイアス アーク内固定, Cycle-Slip時再初期化, Float値推定
基準座標系
・地球固定座標 ITRF
・慣性座標 ICRF (J2000.0平均赤道座標)
・歳差・章動 IAU1976歳差、IAU1980章動＋章動補正 (dpsi/deps)、IERS1996歳差章動
・ERP変動 Subdaily-Variation補正 (Ray model)
・地球回転パラメータ IERS Bulletin B、C04、IGS ERP または推定
・地球中心オフセット 0固定または推定
初期・既知状態量 　
・衛星位置・速度 放送暦または精密暦 (SP3)＋運動モデルによる補間
・衛星時計 放送暦または精密時計 (SP3, RINEX CLK)
・受信機時計 単独測位による推定または精密時計 (RINEX CLK)
・対流圏遅延 A priori対流圏遅延モデル+標準大気
・観測点位置座標 ITRF97、ITRF2000、IGS00, IGb00, IGS推定値(SINEX), GSI推定値, またはユーザ定義
・地球回転パラメータ IGS ERP、IERS Bulletin B, IERS C04
