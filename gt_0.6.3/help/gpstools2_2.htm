観測データ編集設定 Observation Data Editor Settings

項目 操作
L1 Frequency L1搬送波位相周波数 (GHz) を指定します。通常は1.57542GHzに設定してください。
L2 Frequency L2搬送波位相周波数 (GHz) を指定します。通常は1.22760GHzに設定してください。
L1 Code (本バージョンでは使用できません)
Min Elevation Angle 最低仰角 (度) を指定します。
Max Arc Gap 最大アーク間ギャップ (秒) を指定します。観測データにMax Gapより大きなデータ欠損期間があった場合別のアークと判定します。
Min Arc Length 最低アーク長 (秒) を指定します。Min Arc Length未満のアークは無効アークとしてデータを除外します。
Min Arc Points 最低アーク観測データ数 (点) を指定します。Min Arc Points未満の観測データ数のアークは無効アークとしてデータを除外します。
Noise of MW Code Melbourne Webbena Wide Lane線形結合の概算雑音レベル(m)を指定します。
Noise of Ion-Free Code 疑似距離電離層フリー線形結合の概算雑音レベル(m)を指定します。
Slip Threshold of LG 搬送波位相幾何学フリー線形結合飛びによるサイクルスリップ検出閾値 (m) を指定します。指定閾値よりより大きな飛びが検出された場合サイクルスリップと判定します。0を設定した場合、幾何学フリー線形結合によるサイクルスリップ検出を行いません。
Slip Threshold of WL Melbourne Webbena Wide Lane線形結合飛びによるサイクルスリップ検出閾値 (cycle) を指定します。指定閾値より大きな飛びが検出された場合サイクルスリップと判定します。0を設定した場合、MW Wide Lane線形結合によるサイクルスリップ検出を行いません。
Slip Threshold of MP1 MP1線形結合飛びのサイクルスリップ検出閾値 (m) を指定します。指定閾値より大きな飛びが検出された場合サイクルスリップと判定します。0を設定した場合、MP1線形結合によるサイクルスリップ検出を行いません。
Slip Threshold of MP2 MP2線形結合飛びのサイクルスリップ検出閾値 (m) を指定します。指定閾値より大きな飛びが検出された場合サイクルスリップと判定します。0を設定した場合、MP2線形結合によるサイクルスリップ検出を行いません。
Outlier Thres. of MW アウトライアの検出・除外条件を指定します。Melbourne Webbena Wide Lane線形結合の値がアーク内平均より指定値×標準偏差 (σ)以上外れた場合、アウトライアと判定し観測データを除外します。
Moving Ave. Window 1 MW WL, MP1, MP2線形結合によるサイクルスリップ検出時に使用する検出点前後の移動平均、移動標準偏差算出に使用する観測データ数 (点) を指定します。
Moving Ave. Window 2 MW WL, MP1, MP2線形結合によるサイクルスリップ検出時に使用する検出点前後の平均値に使用する観測データ数 (点) を指定します。
Slip Threshold Elevation Weighting サイクルスリップ検出閾値を衛星仰角で重み付けするかどうか指定します。ONにした場合、閾値=指定閾値/sin(仰角)でサイクルリップを検出します。
Repair Steered Clock-Jump Trimble, Ashtech等、定期的に受信機時計飛びが発生する受信機において、時計飛びの検出・修正を行うかどうか指定します。これらの受信機で時計飛び補正を行わない場合、時計飛びをサイクルスリップと誤認することがあります。ONの場合、時計飛びの検出・修正を行い時計飛び前後の擬似距離観測値が連続になるよう疑似距離観測値及び受信時刻タグの値を修正します。OFFの場合時計飛びの検出・修正を行いません。
Repair Cycle-Slip サイクルスリップを検出した場合サイクルスリップを修正するかどうか指定します。ONの場合サイクルスリップ修正を行いサイクルスリップ前後のアークを結合します。OFFの場合、サイクルスリップ修正を行わず、サイクルスリップ前後のアークは別アークとして取り扱われます。
# of Points for Poly Fitting サイクルスリップ修正を行う場合に使用するサイクルスリップ前後の観測データ数 (点) を指定します。
Degree for Poly Fitting サイクルスリップ修正を行う場合に使用する電離層フリー線形結合の多次多項式Fitting次数を指定します。
Separate Arc at Eclipse In/Out 地球食/月食前後のアークを別アークとして扱うか同一アークとして扱うかを指定します。ONの場合別アークとして扱かわれます。　
Phase Mutipath Correction (本バージョンでは使用できません)
Phase MP Prorfile Start/End (本バージョンでは使用できません)
Adustment of Sidereal Day (本バージョンでは使用できません)
Code Multipath Profile (本バージョンでは使用できません)
OK 設定変更を有効にして画面を閉じます。
Cancel 設定変更を無効にして画面を閉じます。
