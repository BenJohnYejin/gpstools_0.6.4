推定・観測モデル設定 Estimation/Measurement Model Settings

項目 操作
Min/Max Elevation Angle 推定に使用する観測データの最低仰角(カットオフ仰角)、最高仰角 (度) を指定します。
Tropospheric Model アプリオリ対流圏遅延モデルを指定します。
・Saastamoinen : Saastamoinenモデル
Tropospheric Mapping Function 対流圏遅延マッピング関数を指定します。
・COSZ : cos zマッピング関数
・NMF : Niell Mapping Function
・GMF : Global Mapping Function
Tropospheric Stochastic Model 対流圏遅延(天頂全遅延)を推定する場合の時間変動モデルを指定します。
・Random-Walk : Random-Walk過程モデル
・Gauss-Marcov : Gauss-Marcov過程モデル
Tropospheric Gradient Model 対流圏遅延勾配モデルを指定します。
・(空白) : 対流圏遅延勾配モデルを使用しません。
・Linear : 1次 (線形) モデル
・Quadratic : 2次モデル
Meas. Noise Elev. Angle Factor 観測雑音を衛星仰角で重み付け係数を指定します。
0の場合重み付けせず観測雑音設定で設定した値がすべての仰角で使用されます。1の場合1/sin(仰角)で重み付けされます。観測雑音ファクタを参照して下さい。
Meas. Noise Factor in Eclipse 食期間及び食後姿勢擾乱期間衛星の観測雑音倍率をBlock II/IIA衛星、Block IIR衛星毎に指定します。
0を指定した場合には指定Block衛星の食期間及び食後姿勢擾乱期間の観測データを除外します。
Process Noise Factor in Eclipse 衛星軌道を推定する場合、食後姿勢擾乱期間の衛星位置/速度プロセスノイズの倍率をBlock II/IIA衛星、Block IIR衛星毎に指定します。
Max Post-Eclipse Manuver 食後姿勢擾乱期間 (秒) を指定します。
Std. Dev. Factor of Initial States 複数パスで推定を行う場合、推定状態量の初期標準偏差として使用する事前パス最終推定状態量標準偏差の倍率を指定します。
0を指定した場合、推定状態量標準偏差は推定パス開始時に初期値にリセットされます。
Std. Dev. Factor of PreDet States 既知状態量の誤差が与えられる場合に、既知状態量標準偏差の倍率を指定します。
0の場合、既知状態量の誤差は0と扱われます。
Max Count of Filter Iteration カルマンフィルタ観測更新則におけるiterationの最大回数を指定します。
通常は1に設定して下さい。
Min Count of Obs for Estimation パラメータ推定を行うための観測データ点数の最小値を指定します。
観測データ点数がこの値未満の場合にはパラメータ推定は行わず欠測として扱われます。
Nutation Model 歳差・章動モデルを指定します。
・IAU1976/1980 : IAU1976歳差/IAU1980章動
・IERS1996 : IERS1996歳差/章動
Filter 使用するフィルタを指定します。
・Standard Kalman : 標準Kalmanフィルタ
Clock Correction by Phase-Bias 衛星/受信機時計推定を行う場合、推定期間中の搬送波位相バイアス推定値平均を使って衛星/受信機時計推定値のオフセット補正を行うか否か指定します。
・OFF : 衛星/受信機時計推定値のオフセット補正を行いません。
・ON : 時計推定値のオフセット補正を行います。
Align Clock to Reference 衛星時計推定を行う場合、推定値を基準時計値と比較し一定以上ずれた推定値を除外するか否か指定します。また推定値のバイアス、ドリフトを補正し絶対値が基準時計に合致するよう修正するか否か指定します。
・OFF : 基準衛星時計による補正を行いません。
・IGS Final : IGS Final時計を基準衛星時計として補正を行います。
・IGS Rapid : IGS Rapid時計を基準衛星時計として補正を行います。
・IGS-CODE : IGS Final時計をCODE時計で補間した時計を基準衛星時計として補正を行います。
Receiver Attitude Model 受信機アンテナの姿勢モデルを指定します。
・Fixed Station : 受信機アンテナが地球に固定された姿勢を取ること仮定します。
・LEO Satellite : 受信機アンテナが移動体に固定されていることを仮定します。移動体の姿勢はZ軸は地心方向、X軸は運動方向に平行であることを仮定します。
Measurement Corrections 観測モデル補正を行うか否か指定します。OFFの場合指定の観測モデル補正を行いません。
・Satellite Antenna Offset : 衛星アンテナ位相中心オフセット/変動
・Receiver Antenna Offset : 受信アンテナ位相中心オフセット/変動
・Relativistic Effects : 相対論効果
・Phase Windup : Phase-Windup効果
・Phase Multipath : 搬送波位相マルチパス
Station Displacement Corrections 観測点位置変位(Site Displacement)による観測局位置座標補正を行うか否か指定します。OFFの場合指定の補正を行いません。
・Solid Earth Tides : 固体地球潮汐
・Ocean Loading : 海洋荷重潮汐
・Pole Tides : 極運動潮汐
・Eliminate Permanent Deform. : 潮汐による永年変位項の除去
ERP Variation Correction 地球回転パラメータの日周以下変動(subdaily variation)の補正を行うか否か指定します。OFFの場合地球回転パラメータ変動補正を行いません。
Meteorological Paraemters 対流圏遅延モデルで使用する気象データの入力先を指定します。
・Standard Atmosphere : 標準大気モデルを使用します。
・JMA GSM Online : 気象庁数値予報モデルデータ(GSM)を読み込みその値を使用します。
・JMA MSM Online : 気象庁数値予報モデルデータ(MSM)を読み込みその値を使用します。
Default MSL Pressure 気象データとして標準大気モデルを使用する場合のデフォルト海面更正気圧(hPa)を指定します。
Default Temp/Rel Humidity 気象データとして標準大気モデルを使用する場合のデフォルト海面気温(C)及び相対湿度(%)を指定します。
OK 設定変更を有効にして画面を閉じます。
Cancel 設定変更を無効にして画面を閉じます。
