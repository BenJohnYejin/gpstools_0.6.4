データディレクトリ/ファイル設定 Data Directory/File Settings

項目 操作
Raw Observation Data RINEX観測データディレクトリを指定します。
直接入力するか[...]ボタンを押して表示されるディレクトリ選択ダイアログでディレクトリパスを選択してください。
ディレクトリパス中のキーワードはディレクトリパス中のキーワードの置換に従って置換されます。
空白が指定された場合には観測データ編集/パラメータ推定設定/実行画面のOutput Directoryに設定されたディレクトリパスが使用されます。(以下ディレクトリパスに関しては同様)
Navigation Message 航法メッセージデータディレクトリを指定します。
Satellite Ephemerides 衛星軌道を既知の値に固定する場合に入力する衛星軌道暦データディレクトリを指定します。
Satellite/Receiver Clocks 衛星時計または受信機時計を既知の値に固定する場合に入力する衛星・受信機時計データのディレクトリを指定します。
Receiver Positions 観測局位置を既知の値に固定する場合に入力する観測局位置データディレクトリを指定します。
Earth Rotation Parameters 地球回転パラメータを既知の値に固定する場合に入力する地球回転パラメータデータディレクトリを指定します。
Tropos/Meteo Parameters 対流圏パラメータ及び気象データを既知の値に固定する場合に入力する対流圏パラメータ/気象データディレクトリを指定します。
Input Estimated Sat Params. 衛星パラメータを推定値に固定する場合に入力する衛星パラメータ推定値のディレクトリを指定します。
Input Estimated Rcv Params. 受信機パラメータを推定値に固定する場合に入力する受信機パラメータ推定値のディレクトリを指定します。
Satellite Antenna PCV 衛星アンテナPCVパラメータファイルを指定します。
ANTEX形式ファイルまたは残差・統計情報表示で出力した衛星アンテナPCVパラメータ(m-file)を指定して下さい。
直接入力するか[...]ボタンを押して表示されるファイル選択ダイアログでファイルを選択してください。[.]ボタンを押すとファイルの内容をテキストエディタで表示することができます。
空白が指定された場合、デフォルトの衛星アンテナPCVパラメータが使用されます。
Receiver Antenna PCV 受信アンテナPCVパラメータファイルを指定します。
IGS_01.PCV形式またはANTEX形式ファイルを指定して下さい。
空白が指定された場合、受信アンテナPCV補正は行いません。
Receiver Position 受信機位置を設定ファイルで指定する場合に受信機位置パラメータファイル(m-file)を指定します。
Ocean Loading Parameter 海洋荷重潮汐係数ファイル(BLQ形式)を指定します。
空白が指定された場合、海洋荷重潮汐による局位置変動補正は行いません。
Excluded Sats/Rcvs 除外衛星/受信機パラメータを指定します。
空白が指定された場合、衛星/受信機を除外しません。
Phase Multipath Profile 搬送波位相マルチパスプロファイルファイルを指定します。
マルチパスプロファイル推定で出力されたマルチパスプロファイルデータを指定して下さい。
空白が指定された場合、搬送波位相マルチパス補正は行いません。
? ディレクトリ・ファイルパス中のキーワード置換に関する情報を表示します。
OK 設定変更を有効として、画面を閉じます。
Cancel 設定変更を無効として、画面を閉じます。

ディレクトリ/ファイルパス中のキーワード置換 Keyword Replacement in Directory or File Settings

キーワード 置換文字列 範囲
%P install directory path 　
%S station code (UPPER case) 　
%s station code (lower case) 　
%G station code (tail 4 chars) 　
%Y year (4 digits) 1970-9999
%y year (2 digits) 00-99
%m month 01-12
%d day 01-31
%n day of year 001-366
%W gpsweek no. 0000-9999
%D day of gpsweek 0,1,2,3,4,5,6
