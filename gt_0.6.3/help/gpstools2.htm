観測データ編集/パラメータ推定設定/実行 Obs Data Editor/Parameter Estimator

項目 操作
Satellites 解析に使用する衛星リストを表示します。
衛星リストを変更する場合は「Satellites...」ボタンを押して表示される衛星リスト編集画面で衛星リストを編集してください。
Receivers 解析に使用する受信機リストを表示します。
受信機リストを変更する場合は「Receivers...」ボタンを押して表示される受信機リスト編集画面で受信機リストを編集してください。
Estimation Start Time 推定開始日時(GPS Time)を設定します。
Estimation End Time 推定終了日時(GPS Time)を設定します。
Estimation Interval 推定時刻間隔(秒)を設定します。
観測データのサンプリング間隔の整数倍を指定してください。
現行バージョンでは1秒以上の値を指定して下さい。
Est Unit/Overlap Time 解析単位時間(hr)及びオーバラップ時間(hr)を設定します。
解析単位時間は推定処理を分割する単位(hr)です。この設定値に従い推定期間が分割され推定結果はこの分割期間単位で格納されます。例えば推定開始/終了日時が2004/1/1 2:30-9:30、推定時刻間隔1分、推定分割単位時間3 hrとした場合、推定期間が2:30-2:59, 3:00-5:59, 6:00-8:59, 9:00-9:30の期間に分割されて推定が実行されます。推定期間の分割を行いたくない場合には十分に大きな値を設定して下さい。
オーバラップ時間は(分割された)推定期間の前後に付加する推定時間です。一般には0を設定して下さい。解析単位時間及びオーバラップ時間の定義を図に示します。
Exec Obs Data Editor 観測データ編集を実行するか否かを設定します。
・ON : 観測データ編集が実行され、編集済観測データが生成されます。観測データ編集に使用される解析パラメータは観測データ編集設定画面で設定したパラメータが使用されます。
・OFF : 観測データ編集は実行されません。事前に観測データ編集を実行し生成された編集済みデータを使って推定を行う場合に選択してください。
Exec Parameter Estimator パラメータ推定の実行パスを設定します。
・OFF : パラメータ推定は実行しません。観測データ編集のみ実行する場合に指定して下さい。
・１pass(F) : 1パス(forward)
・2pass(FB) : 2パス(forward+backward)
・3pass(FBF) : 3パス(forward+backward+forward)
Estimation Strategy パラメータ推定戦略を設定します。
・ZD/PPP : ゼロ差(Undifferenced)またはPPP(精密単独測位)
・DD : 二重差(Double-Differenced)
(現バージョンではDDは正常動作しません)
Est/Fixed Parameters 推定または固定するパラメータを設定します。
ボタンを押して表示される推定/固定パラメータ設定画面で設定を行ってください。
Baselines パラメータ推定戦略が二重差(DD)の場合に使用する基線(Baseline)を指定します。
ボタンを押して表示される基線設定画面で設定を行ってください。
Satellite Clock Model 衛星時計を推定する場合に衛星時計の時間変動モデルを設定します。
・White-Noise : White-Noise過程
・Gauss-Marcov : Gauss-Marcov過程
Receiver Clock Model 受信機時計を推定する場合に受信機時計の時間変動モデルを設定します。
・White-Noise : White-Noise過程
・Gauss-Marcov : Gauss-Marcov過程
Receiver Position Model 受信機位置を推定する場合に受信機位置の時間変動モデルを設定します。
・Static : Static (固定値またはRandom-Walk過程)
・Kinematic : Kinematic (White-Noise過程)
固定観測点のスタティック測位を行う場合には本設定をStaticとした上で受信機位置Process Noise=0と設定してください。
受信機位置の時間変動モデルとしてRandom-Walk過程を選択する場合は、本設定をStaticとした上で受信機位置のProcess Noise設定>0として下さい。
Clock Reference 衛星時計を推定する場合に基準時計受信機を設定します。
指定基準時計受信機の時計誤差=0と仮定した時計誤差相対値が推定されます。
空白に設定した場合には基準時計受信機を指定しません。
Raw Observation Data 入力する観測データ及びファイル分割単位を設定します。
観測データファイル名は以下の[...]で示す形式でつけられている必要があります。
・RINEX : RINEX OBS [rrrrddd0.yyo]
・RINEX (3H) : RINEX OBS (3H) [rrrrdddh.yyo]
・RINEX (1H) : RINEX OBS (1H) [rrrrdddx.yyo]
・RINEX (15min) : RINEX OBS (15min) [rrrrdddxmm.yyo]
(rrrr : 受信機/観測局ID(4桁), ddd : day of year, yy : 年(2桁), h : 1,2,.. (=0:00,3:00,...), x : a,b,c,... (=0:00,1:00,2:00,...), mm : 分)
入力する観測データファイル名が以上の形式に従っていない場合には以上のファイル名に合うようにリネームする必要があります。
観測データファイルはHatanaka-Compression, Compress, Gzip圧縮されていてもかまいません。
圧縮ファイルの場合にはそれに応じたファイル拡張子がつけられている必要があります。
(例 : *.yyo.Z, *.yyo.gz, *.yyd, *.yyd.Z, *.yyd.gz)
Navigation Message 入力する航法メッセージ及びファイル分割単位を設定します。
航法メッセージファイル名は以下の[...]で示す形式でつけられている必要があります。
・Combined : IGS Combined RINEX NAV (daily) [brdcddd0.yyn]
・RINEX : RINEX NAV (daily) [rrrrddd0.yyn]
・RINEX (3hr) : RINEX NAV (3H) [rrrrdddh.yyn]
・RINEX (1hr) : RINEX NAV (1H) [rrrrdddx.yyn]
・RINEX (15min) : RINEX NAV (15min) [rrrrdddxmm.yyn]
入力する航法メッセージファイル名が以上の形式に従っていない場合には以上のファイル名に合うようにリネームする必要があります。
Obs Data Editor Settings 観測データ編集のパラメータを設定します。
ボタンを押して表示される観測データ編集設定画面で設定してください。
Estimation/Meas. Model 推定・観測モデルパラメータを設定します。
ボタンを押して表示される推定・観測モデル設定画面で設定を行ってください。
Measurement Noises 観測雑音を設定します。
ボタンを押して表示される観測雑音設定画面で設定を行ってください。　
A Priori Std. Deviation 推定状態量の初期値誤差を設定します。
ボタンを押して表示される推定状態量初期誤差設定画面で設定を行ってください。
Process Noises 推定状態量のプロセスノイズを設定します。
ボタンを押して表示される推定状態量プロセスノイズ設定画面で設定を行ってください。
Satellite Orbit Model 衛星軌道を推定する場合、衛星運動モデルパラメータを設定します。
ボタンを押して表示される衛星運動モデル設定画面で設定を行ってください。
Quality Control Settlings 品質管理パラメータを設定します。
ボタンを押して表示される品質管理設定画面で設定を行ってください。
Data Directories/Files データディレクトリ及びファイルを設定します。
ボタンを押して表示されるデータディレクトリ/ファイル設定画面で設定を行ってください。
Clean Observation Data Directory 編集済観測データファイル出力ディレクトリパスを設定します。
ディレクトリパス中にキーワードが含まれる場合、ディレクトリパス中のキーワード置換に従い置換されます。指定ディレクトリが存在しない場合、自動的にディレクトリが生成されます。
本設定を空白にした場合、Output Directoryで指定したディレクトリが使用されます。
Output Directory 解析結果データ出力ディレクトリパスを設定します。
ディレクトリパス中にキーワードが含まれる場合、ディレクトリパス中のキーワード置換に従い置換されます。指定ディレクトリが存在しない場合、自動的にディレクトリが生成されます。
File 　
　 Files... ファイル管理/操作ツールを起動します。
Download... データ/プロダクトダウンローダを起動します。
Execute Batch... バッチ推定画面を表示します。
Load Setting... 解析パラメータファイルを読み込みます。
表示されるファイル選択ダイアログで保存する解析パラメータファイルを指定してください。
読み込む解析パラメータファイルにはSave Settingメニューで保存した解析パラメータ、解析実行時に実行ヒストリディレクトリ下に解析ヒストリ (<install path>\history\prm_*.mat>) としてに保存された解析パラメータが使用できます。
Save Setting... 設定した解析パラメータに名前をつけて解析パラメータファイルに保存します。
表示されるファイル保存ダイアログで保存する解析パラメータファイルパスを指定してください。
Log 観測データ編集/パラメータ推定実行ログを表示します。
Plot 　
　 Observation Data 観測データ表示画面により観測データ/編集済観測データの内容を表示します。
Satellite Orbit... 衛星軌道表示画面により推定結果を表示します。
Satellite Clock... 衛星/受信機時計表示画面により推定結果を表示します。
Receiver Position... 受信機位置表示画面により推定結果を表示します。
Receiver Clock... 衛星/受信機時計表示画面により推定結果を表示します。
Tropospheric Parameters... 対流圏パラメータ表示画面により推定結果を表示します。
Residuals/Statistics... 残差/統計情報表示画面により推定残差を表示します。
Satellite Track/Station Pos... 衛星直下点軌跡/受信機位置表示画面を表示します。
Execute/Abort 設定した解析パラメータを使って解析を実行します。
解析実行途中で実行を中止する場合はAbortボタンを押して下さい。
Close 観測データ編集/パラメータ推定設定/実行画面を閉じます。

衛星リスト編集 Satellite List Editor

項目 操作
Selected Satellites 星リストを表示します。
(Selectable Satellites) 選択可能衛星リストを表示します。
<< 選択可能衛星リストで選択された衛星を選択済み衛星リストに追加します。
>> 衛星リスト中の選択された衛星を衛星リストから削除します。

ファイルは1行1衛星で記述されたテキストファイルまたは解析パラメータファイル(prm_*.mat)を指定して下さい。
Save... 衛星リストをテキストファイルとして保存します。
ボタンを押して表示されるファイル保存ダイアログで保存先ファイルを指定して下さい。
List... 衛星リスト登録/呼び出し画面を表示します。

選択可能衛星リストに含まれない衛星を衛星リストに追加する場合等に使用して下さい

(受信機リスト編集 - リスト表示)


(受信機リスト編集 - 地図表示)

項目 操作
Selected Receivers 受信機リストを表示します。
(Selectable Receivers Group) 選択可能受信機リストグループを指定します。
(Selectable Receivers) 選択可能受信機リストを表示します。
<< 選択可能受信機リストで選択された受信機を受信機リストに追加します。
>> 選択された受信機を選択済み受信機リストから削除します。
Clear 受信機リストをクリアします。
Edit... 受信機定義ファイルをテキストエディタで表示します。
選択可能受信機リストの内容を変更する場合は受信機定義ファイル (<install path>\data\prm_gpsrcvs.txt) をエディタ等で修正してください。
Load... 受信機リストをファイルから読み込みます。
ボタンを押して表示されるファイル選択ダイアログで読み込むファイルを指定して下さい。
ファイルには1行1受信機で記述されたテキストファイルまたは解析パラメータファイル(prm_*.mat)を指定して下さい。
Save... 受信機リストをテキストファイルとして保存します。
ボタンを押して表示されるファイル保存ダイアログで保存先ファイルを指定して下さい。
List... 受信機リスト登録/呼び出し画面を表示します。

選択可能受信機リストに含まれない受信機を受信機リストに追加する場合等に使用して下さい。
MapView/ListView... 受信機リストのリスト表示/地図表示を切り替えます。
地図表示の場合選択された受信機の位置が地図上に赤点●で表示されます。
・地図上を右ボタンでドラッグすることにより矩形領域に含まれる受信機を選択することができます。
・地図上を左ボタンでドラッグすることにより矩形領域に含まれる受信機を追加することができます。
・地図上でダブルクリックすることにより指定位置を地図の中心に移動することができます。
・スクロールバーを操作することにより地図の縮尺を変更することができます。
OK 設定変更を有効にして画面を閉じます。

衛星/受信機リスト登録/呼び出し Add/Set Satellite/Receiver List as List Name

項目 操作
Satellite/Receiver List 登録済み衛星/受信機リストを表示します。一覧から登録済み衛星/受信機リストを選択し「OK」ボタンを押すか、リストをダブルクリックすることにより、選択した登録済み衛星/受信機リストが呼び出され、選択済み衛星/受信機リストに入力されます。
(Newlist Input) 新規登録する衛星/受信機リスト名を入力します。
Add/Set (Newlist Input)に入力されたリスト名で選択済み衛星/受信機リストを登録します。新規リスト名の場合、登録済み衛星/受信機リストに追加します。既存リスト名の場合は既存リスト名の内容が書き換えられます。
Delete 選択した登録済み衛星/受信機リストを登録済み衛星/受信機リスト一覧から削除します。
OK 設定変更を有効として画面を閉じ、選択した登録済み衛星/受信機リストを呼び出します。
Cancel 設定変更を無効として画面を閉じます。

バッチ推定 Execute Batch Estimation

項目 操作
Estimation Start Time/End Time (GPST) 解析パラメータリストの推定開始/終了時刻を変更してバッチ推定を行う場合、チェックボックスをONにして推定開始/終了時刻を指定して下さい。チェックボックスがOFFの場合には個々の解析パラメータで指定された推定開始/終了時刻が使用されます。
Receivers 解析パラメータリストの受信機リストを変更してバッチ推定を行う場合、チェックボックスをONにして受信機リストを指定して下さい。チェックボックスがOFFの場合には個々の解析パラメータで指定された受信機リストが使用されます。
Output Directory 解析パラメータリストの解析結果出力ディレクトリを変更してバッチ推定を行う場合、チェックボックスをONにして解析結果出力ディレクトリを入力して下さい。チェックボックスがOFFの場合は個々の解析パラメータで指定された出力ディレクトリが使用されます。
Processing Parameters 解析パラメータリストを一覧表示します。Add...ボタン、Deleteボタンを使用し解析パラメータファイルパスを入力して下さい。
バッチ推定にあたって指定した解析パラメータリストは一覧の上から下に向けて順次使用されます。
バッチ推定実行中には実行中の解析パラメータが反転表示されます。
Add... 解析パラメータリストに解析パラメータを追加します。ボタンを押して表示されるファイル選択ダイアログで追加したい解析パラメータファイルパス(prm_*.mat)を指定して下さい。追加する解析パラメータは観測データ編集/パラメータ推定設定/実行画面のメニュー「File」「Save Setting...」を使って保存した解析パラメータ、解析実行時に実行ヒストリファイル (<install_Directory>\history\prm_*.mat> に保存された解析パラメータが使用できます。
Delete 解析パラメータリストから選択された解析パラメータを削除します。
Log バッチ推定実行ログを表示します。
Execute/Abort 指定された解析パラメータリストを使用してバッチ推定を実行します。
実行途中で実行を中止する場合にはAbortボタンを押してください。
Close バッチ推定画面を閉じます。
