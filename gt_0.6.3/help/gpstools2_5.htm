衛星運動モデル設定 Satellite Orbit Model Settings

項目 操作
Max Degree of Geopotential 地球重力モデルの最大次数を指定します。
Solar/Planetary Potentials 他天体重力モデルを指定します。
・OFF : 他天体重力を考慮しません。
・Sun/Moon : 月及び太陽重力を考慮します。
Solar Radiation Pressure Model 太陽輻射圧モデルを指定します。
・OFF : 太陽輻射圧を考慮しません。
・Simple : 単純モデル
・ROCK4 : ROCK4/42モデル
・GSPM.04 : GSPM.04bモデル+scale,ybias推定
・GSPM.04+A/C : GSPM.04bモデル+scale,ybias,along-trk,cross-trk加速度推定
・CODE(D/Y/B) : CODEモデル+3パラメータ(D0,Y0,B0)推定
・CODE(D/Y/B/Z) : CODEモデル+4パラメータ(D0,Y0,B0,Z0)推定
・CODE(D/Y/B/Z/X) : CODEモデル+6パラメータ(D0,Y0,B0,Z0,X10,X30)推定
Eclipse Model 食モデルを指定します。
・Cylindric : 円筒モデル
・Penumbra/Unmra : 本影・半影モデル
Atmospheric Drag Model 大気抵抗モデルを指定します。
・OFF : 大気抵抗を考慮しません。
Earth Tides 地球潮汐効果による地球重力補正を行うか否か指定します。
・OFF : 地球潮汐効果の補正を行いません。
・Solid Earth Tide : 固体地球潮汐補正
・Solid/Ocean Tide : 固体地球潮汐、海洋潮汐、極運動潮汐補正
Relativistic Effects 相対論効果による加速度補正を行うか否か指定します。
・OFF : 相対論効果による加速度補正を行いません。
・IERS : IERS Conventions 1996補正
Thrust Force スラスターによる加速度補正を行うか否か指定します。
・OFF : スラスターによる加速度補正を行わない。
Integration Step 数値積分時間間隔(秒)を指定します。
OK 設定変更を有効にして画面を閉じます。
Cancel 設定変更を無効にして画面を閉じます。
